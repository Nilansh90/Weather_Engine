        deg = degrees % 360
        bins = ['N', 'NE', 'E', 'SE', 'S', 'SW', 'W', 'NW']
        return bins[int((deg + 22.5) % 360 / 45)]

j)Strategised for Light GBM model ->
General strategy and motifs ->

1)The model is a full weather engine and hence the model should be able to predict all the
weather parameters given the date and city_id. Hence, the model will be trained on all the features and will predict all
 the features. The model will be trained on the train set and validated on the validation set. The test set will be used
 for final evaluation of the model.
2)We train one model for each feature to be predicted. Hence, we will have 7 models in total for the 7 features to be predicted.
3)The features to be predicted are -> temp_max,temp_min,wind_speed_max,wind_dir_dominant,precip_sum,precip_hours,weather_code
4)The features to be used for prediction are -> year,month_sin,month_cos,day,day_of_week,city_id
